Advanced Custom Fields (ACF): converts Cyrillic characters in field name to Latin characters.

== Description ==

Add-On plugin for Advanced Custom Fields (ACF) that converts Cyrillic characters in field name to Latin characters.

== Installation ==

1. Upload `acf-rustolat` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
